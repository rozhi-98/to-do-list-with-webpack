# 📖 [To-do-list-app] <a name="about-project"></a>

**[To-do-list-app]** is a basic website that allows users to right down our daily tasks in the form of a list. We will achieve that by using JavaScript objects and arrays. We will also need to dynamically modify the DOM and add basic events.👌

## 🛠 Built With <a name="built-with"></a>
- HTML
- JavaScript
- Javascript DOM
- Webpack
- CSS

### Key Features <a name="key-features"></a>

- **[Remove_tasks]**
- **[Save_in_local_storage]**
- **[Remove-all-finished-tasks-at-once]**

## 💻 Getting Started <a name="getting-started"></a>

To get a local copy up and running follow these simple steps.

### Prerequisites

- Use `git@github.com:rozhi-98/to-do-list-with-webpack.git` to clone the repository using the SSH link into your local machine

### Setup

- `cd` into your local repository

### Install

- `npm install` - Installs the necessary dependencies

### Usage

- Open the index file and enjoy
